# Support Requirements System (MVP)

Simple internal support requirements management MVP.

Quick start

1. Install dependencies

```bash
npm install
```

2. Start server

```bash
npm start
```

3. Open http://localhost:3000

Notes

- DB file `data.db` will be created in the project root.
- This is an MVP: authentication, file attachments, and advanced features are intentionally omitted but can be added.
